### 2.2 핵심 알고리즘

#### 중복 질문 방지 (하이브리드 접근)
- OpenAI 임베딩으로 의미적 유사성 확인 (0.85 임계값)
- 키워드 기반 폴백 메커니즘
- 최대 2회 재시도 + temperature 점진 증가

#### 프롬프트 시스템 (다층 구조)
- `NCS_ANALYSIS_SYSTEM_PROMPT`: 공기업 NCS 채용 분석 핵심
- `FALLBACK_SYSTEM_PROMPT`: 표준 면접관 페르소나
- `PROMPT_PRESETS`: 4가지 면접 스타일 (기본/압박/친절/검증)
- 2개 페르소나: 인사팀장(10년차), 실무차장(7년차)

#### 평가 시스템
| 구분 | 인성 면접 | 직무 면접 |
|------|----------|----------|
| 평가 항목 | 조직 적합성, 윤리성, 팀워크 | 직무 적합성, 문제해결력, 전문성 |
| 구조 분석 | STAR (Action 가중치 높음) | 논리 4요소 (Method/Tool/Data/Risk) |
| 감점 기준 | 추상적 표현 | 방법론/도구 언급 없음 |
| 과락 조건 | 윤리성 60점 미만 | - |

| 영역 | 점수 | 평가 |

- `embeddings.ts` - 임베딩 기반 유사도 검사
- NCS 분류 체계 기반 직렬 매핑
